## Hadoop
- 대량의 데이터의 저장 및 처리를 관리하는 Java 기반 오픈소스 프레임워크
- 여러개의 컴퓨터를 하나로 묶어 대용량 데이터 처리
    - 분산처리시스템(HDFS) : 수천대의 분산된 장비에 대용량 파일을 저장할 수 있는 기능
    - 맵리듀스 플랫폼(MapReduce) : 저장된 분산 파일을 분산된 서버의 CPU와 메모리 자원을 이용하여 빠르게 분석
